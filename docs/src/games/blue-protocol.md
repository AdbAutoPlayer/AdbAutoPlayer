# Blue Protocol Star Resonance

## Bot Features
AdbAutoPlayer provides automation for various activities in Blue Protocol Star Resonance:

* **Fishing:** Automate the fishing minigame to collect resources.
* **Simple Gathering:** Automatically gather resources from the world.
* **Wondrous Tag:** Automate the Wondrous Tag activities.
* **Devourer:** Help with Devourer related tasks.
