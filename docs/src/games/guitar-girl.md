# Guitar Girl

## Bot Features
AdbAutoPlayer helps you with the tapping in Guitar Girl:

* **Auto Tapper:** Automatically taps to generate likes.
* **Skill Activation:** Uses skills as they become available.
* **Daily Collection:** Collects daily rewards and bonuses.
